# Fullstack demo
This is a small demo for fullstack web page using **Flask** and **React**.
- React (TS)
- Vite
- Flask
- RESTFul API


# Build
## Frontend
**Create React project**
```
npm create vite@latest
```
**Build distribution**
```
cd <FRONTEND_NAME>
npm run build 
```

## Backend

```
pip install -r requirements.txt
```

```
python backend/app.py
```
